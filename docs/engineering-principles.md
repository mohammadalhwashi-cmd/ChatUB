# Engineering Principles

This repository follows the public portfolio standard used across Abdulelah Alkhathami's flagship AI projects.

## Principles

- Ground AI responses in trusted domain content where possible.
- Keep the first system version simple enough to evaluate.
- Make data sources, retrieval behavior, and generation boundaries explicit.
- Document model limitations before claiming user impact.
- Prefer repeatable setup and clear architecture over impressive but unclear demos.

## Applied Here

- Local university FAQ data anchors answer generation.
- Semantic search selects context before the Ollama response layer.
- The README describes generated artifacts and evaluation gaps openly.
- Future work prioritizes citations, offline embeddings, and answer-quality testing.
